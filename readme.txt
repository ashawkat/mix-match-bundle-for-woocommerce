== Installation ==

= Automatic Installation =

1. Log into your WordPress admin panel
2. Go to **Plugins** > **Add New**
3. Search for "Mix & Match Bundle"
4. Click **Install Now** and then **Activate**
5. Navigate to **Mix & Match** in the admin menu
6. Create your first bundle!

= Manual Installation =

1. Download the plugin ZIP file
2. Log into WordPress admin panel
3. Go to **Plugins** > **Add New** > **Upload Plugin**
4. Choose the ZIP file and click **Install Now**
5. Click **Activate Plugin**
6. Navigate to **Mix & Match** in the admin menu

= After Activation =

1. Go to **Mix & Match** in your WordPress admin menu
2. Click **Add New Bundle** or create one inline
3. Enter bundle name and description
4. Search and select products to include
5. Add discount tiers (quantity and discount percentage)
6. Customize colors and text (optional)
7. Save the bundle
8. Copy the shortcode (e.g., `[mmb_bundle id="1"]`)
9. Paste shortcode on any page or post
10. Preview and test your bundle!

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes! This plugin is an extension for WooCommerce and will not work without it. WooCommerce must be installed and active before you can use Mix & Match Bundle. If you try to activate the plugin without WooCommerce, you'll see an error message with a link to install WooCommerce.

= Does this work with variable products? =

Yes! The plugin fully supports WooCommerce variable products. Customers can select specific variations using dropdown menus before adding to their bundle.

= Can I create multiple bundles? =

Absolutely! Create unlimited bundles with different products, discount structures, and designs. Each bundle gets its own unique shortcode.

= How do I display a bundle on my site? =

Use the shortcode `[mmb_bundle id="X"]` where X is your bundle ID. You can find the shortcode with a copy button in the admin bundle list.

= Can I customize the appearance? =

Yes! Customize 5 color schemes, button text, progress bar title, and control which elements are visible. The plugin also uses CSS variables for advanced customization.

= Does the discount apply automatically? =

Yes! Discounts are calculated in real-time as customers select products and applied automatically when they add the bundle to their cart.

= Is this compatible with my theme? =

Yes! The plugin is designed to work with any properly coded WordPress theme. It uses standard WooCommerce styles and modern CSS practices.

= Will this work with my sidecart plugin? =

Yes! The plugin integrates with popular sidecart plugins like FunnelKit, WooCommerce Side Cart, and most theme-based sidecarts. You can also configure it to redirect to the cart page if preferred.

= Is it compatible with HPOS? =

Yes! The plugin fully supports WooCommerce High-Performance Order Storage (HPOS/Custom Order Tables) for better performance.

= Can I translate the plugin? =

Yes! The plugin is fully internationalized with 28 pre-configured language files. You can translate using Poedit, Loco Translate, or any gettext editor.

= Does it work on mobile devices? =

Yes! The plugin is fully responsive and includes a special sticky mobile cart footer for an optimal mobile shopping experience.

= How are discounts calculated? =

Discounts are based on the total quantity of items in the bundle. If you set a tier for 3 items at 15% off, customers get 15% off their entire bundle when they select 3 or more items.

= Can I use this with subscription products? =

The plugin works with simple and variable products. Subscription product support depends on your subscription plugin's cart integration.

= What happens if I deactivate the plugin? =

Bundles created with the plugin will no longer display, but your products remain unaffected. Bundle-related data is safely stored and restored if you reactivate.

= Can I control the order products appear on the frontend? =

Yes! NEW FEATURE: We've added an intuitive drag-and-drop interface. After selecting products, they appear in a "Selected Products" section where you can drag them up or down to reorder. Products will display on the frontend in exactly the order you set. This is perfect for featuring bestsellers first or arranging products strategically!

= Can I limit product quantities? =

Currently, customers can select any quantity in Quantity Mode. Selection Mode limits to one of each product. Quantity limits per product are on the roadmap.

= Is there a premium version? =

This is a full-featured plugin. We may release additional features based on user feedback. Follow us for updates!
